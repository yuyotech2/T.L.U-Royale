# T.L.U Royale - V.2.0.0

A high-performance, browser-based 2D Battle Royale experience built with Vanilla JavaScript and HTML5 Canvas.

![Game Banner](https://via.placeholder.com/1200x400.png?text=T.L.U+Royale+Battle+Island)

## 🎮 Game Features

### 🌍 Massive Map Options
- **Standard Mode**: 6,000m x 6,000m island with 16 players.
- **Path to Pro (Competitive)**: An absolute behemoth 15,000m x 15,000m island.
- **Dynamic POIs**: Explore locations like Prominence Peak, Battle Bayou, and more.

### 👥 Multiplayer & Social
- **Lobby System**: Host or join rooms using Peer-to-Peer (PeerJS) technology. No central server required!
- **Party Simulator**: Add AI teammates to your squad for Duo and Squad modes.
- **Squad Tactics**: AI teammates will follow you and defend you against enemy squads.

### 🏆 Progression System
- **Leveling**: Gain XP and Level Up through the "Path to Pro" mode.
- **Career Stats**: Tracks your total wins and career eliminations.
- **Kill Leader**: Claim the golden crown 👑 by having the most kills in a match.

### 👕 Customization (Locker)
- Choose from premium skins: **Original Blue, Crimson, Midas (Gold), Nebula, Forest,** and **Ghost**.
- Tactical visual feedback and premium weapon icons (AK-47, SMG, Pump Shotgun).

## 🚀 How to Play

1. **GitHub Pages**: You can play this game directly in your browser if hosted via GitHub Pages!
2. **Local**: Simply open `index.html` in any modern web browser.

### Controls
- **WASD**: Move
- **Mouse**: Aim & Shoot
- **Scroll/1-5**: Switch Weapons
- **SPACE**: Drop from Plane / Use Glider
- **E**: Interact / Open Crates
- **Q**: Drop held weapon
- **TAB**: Toggle Large Map

## 🛠️ Technical Details
- Built with **Vanilla JS** and **HTML5 Canvas**.
- Uses **PeerJS** for serverless P2P multiplayer logic.
- **Local Storage**: Saves your nickname, wins, kills, and skins automatically.
